# infinite_note
### 문제와 답을 입력하여 정답/오답을 관리하는 콘솔 프로그램

# 설치
- [https://pypi.org/project/infinite_note](https://pypi.org/project/infinite_note)

# 사용법
 - inf-note-input = 문제 입력
 - inf-note-count = 입력된 문제 갯수 확인
 - inf-note-answer = 입력된 문제 출력 및 답안 저장
